To get the application download files from repo

To run the application you need to have JRE installed onto your computer

How to install JRE: https://docs.oracle.com/goldengate/1212/gg-winux/GDRAD/java.htm#BGBFJHAB

To run application: 
1. Open command prompt
2. Get to the Application.java folder
3. Compile Application.java
4. Run Application.class
